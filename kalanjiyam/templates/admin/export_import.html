Export All Projects

Export Includes:

  • • Project metadata (title, author, description, etc.)
  • • All pages with their order and status
  • • Complete revision history for each page
  • • Translations (if any exist)
  • • Discussion boards, threads, and posts
  • • Original PDF files
  • • Page images (JPG files)

Import Books

Import Single Project

Import a book from an exported ZIP file.

Import All Projects

Import multiple books from an exported ZIP file.

⚠️ Important Notes:

  • • Importing will create new users for any authors that don't exist
  • • Users will have placeholder passwords that should be changed
  • • Projects with duplicate slugs will be rejected
  • • Import process is irreversible - backup your database first
  • • Large imports may take several minutes
